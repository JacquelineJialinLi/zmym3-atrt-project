# 02_Normalization_HVG — Normalization & Highly Variable Gene Figures

Figures generated by `notebooks/02_Normalization_HVG.ipynb`.

Typical contents:
- Pre- vs. post-normalization count distributions
- Mean-dispersion plot from `sc.pp.highly_variable_genes` (Seurat method)
- Top-2,000 HVG selection visualization
- Optional: ZMYM3's location in the mean-dispersion space (note: ZMYM3 is not among the top 2,000 HVGs, consistent with its low and sparse expression).

Counts are normalized to 10,000 per nucleus (CP10K) and log1p-transformed prior to HVG selection.
