# Data

The raw and processed data files are not stored in this repository due to size constraints. Below are instructions for obtaining the data used in this analysis.

## Expected directory structure

```
data/
├── README.md                          <- This file
├── snrnaseq/                          <- Paassen et al. snRNA-seq data
│   └── (downloaded files)
└── depmap/                            <- DepMap 26Q1 release
    ├── CRISPRGeneEffect.csv
    ├── Model.csv
    └── OmicsExpressionProteinCodingGenesTPMLogp1.csv
```

---

## 1. snRNA-seq data — Paassen et al. (2025)

- **GEO accession:** [GSE283842](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE283842)
- **Citation:** Blanco-Carmona, E., Paassen, I., He, J., et al. *Neuro-Oncology* **27**, 3260–3275 (2025). doi: [10.1093/neuonc/noaf179](https://doi.org/10.1093/neuonc/noaf179)
- **Description:** 10x Chromium multiome snRNA-seq from 7 AT/RT patient samples (Maxima cohort), with author-deposited cell-level metadata including QC annotations, Scrublet doublet scores, UMAP coordinates, cell type labels, and AT/RT subgroup assignments.

**Download:**
1. Visit the GEO page above.
2. Download the supplementary files (processed count matrices and metadata tables).
3. Place all files in `data/snrnaseq/`.

The raw FASTQ files were not used in this analysis — the deposited count matrices were sufficient.

---

## 2. DepMap data (26Q1 release)

- **Source:** [DepMap Portal — Downloads](https://depmap.org/portal/data_page/?tab=allData)
- **Release:** DepMap Public 26Q1
- **Citation:** Meyers, R.M., et al. *Nature Genetics* **49**, 1779–1784 (2017). doi: [10.1038/ng.3984](https://doi.org/10.1038/ng.3984)

**Files needed:**

| File | Description |
|------|-------------|
| `CRISPRGeneEffect.csv` | Chronos-corrected CRISPR gene-effect scores (1,208 cell lines × 18,531 genes) |
| `Model.csv` | Cell-line metadata, including OncotreeSubtype classifications |
| `OmicsExpressionProteinCodingGenesTPMLogp1.csv` | CCLE bulk RNA-seq expression (log2[TPM+1]) |

**Download:**
1. Go to the DepMap Portal data downloads page.
2. Select the **26Q1** release.
3. Download the three files listed above and place them in `data/depmap/`.

---

## Notes

- All data sources are publicly available and free for academic use.
- The `.gitignore` file is configured to exclude all files in `data/snrnaseq/` and `data/depmap/` from version control, so you can download data into these folders without cluttering the repository.
- Processed intermediate files (e.g., `.h5ad` AnnData objects produced by the notebooks) are also excluded from the repo. They are regenerated by re-running the notebooks in order.
