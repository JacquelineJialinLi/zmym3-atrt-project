# 01_QC — Quality Control Figures

Figures generated by `notebooks/01_QC.ipynb`.

Typical contents:
- Distributions of detected genes per nucleus (n_genes)
- Mitochondrial fraction distributions
- Scrublet doublet score distributions
- Pre- vs. post-filtering counts per sample
- QC scatter plots (n_genes vs. % mito, etc.)

After applying QC thresholds (200–6,000 genes per nucleus, <20% mitochondrial fraction, Scrublet score ≤ 0.25), 9,604 of 9,703 deposited nuclei were retained (98.9%).
