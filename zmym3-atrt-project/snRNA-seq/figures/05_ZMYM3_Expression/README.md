# 05_ZMYM3_Expression — ZMYM3 Expression Analysis Figures

Figures generated by `notebooks/05_ZMYM3_Expression.ipynb`. **These are the main snRNA-seq results figures from the paper.**

Typical contents:
- ZMYM3 UMAP feature plot (normalized log1p expression)
- **Figure 5** — Violin plot of ZMYM3 expression across annotated cell types
- **Figure 6** — Violin/box plots of ZMYM3 expression in tumor cells stratified by AT/RT subgroup (TYR / SHH / MYC)

**Key statistical results:**
- Tumor vs. non-malignant comparisons (Wilcoxon rank-sum, BH-FDR corrected, one-sided tumor > reference):
  - vs. microglia: FDR < 0.001 (tumor higher)
  - vs. endothelial: FDR = 0.043
  - vs. pericytes: FDR = 0.043
  - vs. astrocytes: FDR = 0.095 (n.s.)
  - vs. neurons: FDR = 0.993 (tumor *lower*)
- Subgroup comparison (Kruskal-Wallis): H = 10.43, p = 5.43 × 10⁻³
  - ATRT-TYR vs. ATRT-MYC: FDR = 0.025
  - ATRT-TYR vs. ATRT-SHH: FDR = 0.016
  - ATRT-MYC vs. ATRT-SHH: FDR = 0.706 (n.s.)

**Bottom line:** ZMYM3 is expressed at low levels (<11% of nuclei) across all cell types and subgroups, with no tumor-specific enrichment. This argues for a non-oncogene addiction model rather than a transcriptionally selective target.
